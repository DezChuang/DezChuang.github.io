Question Set

1.為什麼我們需要 CSS 預處理器？沒有 CSS 預處理器的話會怎樣嗎？

使用CSS預處理器能夠在往後處理更複雜的大型專案時，能有效率地偵錯與開發。

2.在那三套裡面，你為什麼選擇了現在這一套？理由是什麼？

選擇使用sass，雖然有點想偷懶學看起來比較平易近人的Stylus，但因為這篇Roadmap推薦學習sass，實際上為什麼還需要慢慢摸索。

Troubleshooting

  • .container無法用margin: 0 auto置中 [Fixed]

原因: .container需指定width為1000px，若指定為72%會有內容元素無法對齊的問題，要做RWD可能還是需使用media query

  • 目前使用Prepros來編譯sass，但似乎有更多有效率地方法如Compass或gulp等等，還需要再學習摸索。
